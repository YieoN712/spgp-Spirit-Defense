## 게임 컨셉
### High Concept
  타일 기반 맵 위에서 유닛을 배치하여 적의 이동 경로를 방어하는 **전략형 타워 디펜스 게임**
### 핵심 기술
  타일 위에 유닛 배치  
  지정된 경로(path)로 이동하는 몬스터  
  자동으로 범위내의 몬스터를 인지하는 유닛  
  코스트(자원)를 사용하여 유닛 생성  
  모든 적을 잡으면 승리 / 일정 수 이상 통과 시 패배  

## 개발 범위
### 시스템 구성
  Scene 시스템 (타이틀 / 게임 / 결과 화면)  
  타일맵 시스템 (Tiled Map Editor 연동)  
  몬스터 이동 경로 시스템  
  유닛 배치 시스템  
### 게임 로직
  거리 기반 자동 공격 시스템  
  충돌 처리  
  몬스터 생성기 (웨이브 시스템)  
  코스트 시스템  
### UI
  시작 화면  
  메인 로비  
  게임 UI (코스트, 유닛 선택)  
  결과 화면 (승리/패배)  
### 유닛 & 몬스터
  유닛: 3종 이상  
  몬스터: 3종 이상  
  맵: 1개 이상  
  레벨: 1개 이상
